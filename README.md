一个基于 Pygame 的轻量级视觉小说（Galgame）引擎，让你用简单的脚本写出自己的故事。  
项目中包含一些示例


**特性**

- 逐句对话
- 选项分支切换
- 存档 / 读档
- 自定义UI


**环境要求**
- Python 3.6+
- Pygame 2.5+

*项目在windows11上可以正常运行,linux等还未测试*


**快速开始**
```
pip install -r requirements.txt
python main.py
```



**制作自己的GALGAME**

按照[规则](对话架构介绍.md)修改text.py
